# YouTube Hide Nav Ext 🎥

## Overview

YouTube Hide Nav Ext is a lightweight browser extension for Chrome and Firefox designed to enhance your YouTube viewing experience. This extension discreetly addresses the frustration of accidentally clicking on YouTube's previous and next navigation buttons.

## Features

- 📵 Discreetly hides YouTube's ⏮️ previous and ⏭️ next navigation buttons.
- 🖥️ Provides a clutter-free interface for an immersive video-watching experience.
